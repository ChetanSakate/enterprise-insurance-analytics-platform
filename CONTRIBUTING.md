# Contributing

Thank you for your interest in contributing to the Enterprise Insurance Analytics Platform.

## How to Contribute

- Report issues or bugs.
- Suggest new analytical features.
- Improve SQL queries.
- Enhance dashboard visualizations.
- Improve documentation.

## Development Guidelines

- Maintain consistent SQL formatting.
- Preserve dimensional modeling principles.
- Keep business logic separated from visualization logic.
- Document all significant changes.

## Reporting Issues

Please include:

- Description
- Expected behavior
- Steps to reproduce
- Suggested improvement

Thank you for helping improve this project.
